# DüğünDefteri - Takı & Düğün Yönetim Platformu

## Proje Özeti
Doğu ve Güneydoğu Anadolu düğün kültürüne özel, tamamen ücretsiz ve reklam gelir modeliyle çalışan SaaS mobil uygulaması.

## Temel Özellikler
- **Takı Muhasebesi**: Altın/Döviz/TL bazlı takı takibi + otomatik kur hesaplama
- **Akıllı Takvim**: Sezonluk düğün yoğunluğu takibi + çakışma uyarıları
- **Geri Dönüş Listesi**: Geçmiş düğün katılımlarından otomatik davetli listesi
- **Dijital Kayıt**: Anlık "kim ne taktı" notu alma

## Tech Stack
- **Mobile**: Flutter
- **Backend**: Next.js (API Routes)
- **Database**: PostgreSQL (Supabase)
- **Offline**: Hive/SQLite (local storage)
